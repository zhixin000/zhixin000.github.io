IoT-Based Environmental Monitoring System

Category: IoT · Tools: LoRaWAN, Wi-Fi, Node-RED · Year: 2024
This IoT system measures energy consumption, temperature, humidity, and occupancy using a hybrid communication model that prioritizes LoRaWAN for long-range, low-power transmission.
The system achieved a 25% improvement in power efficiency and maintained a low latency of 2.3 seconds. Collected data were stored in a time-series database and visualized using Node-RED dashboards.
